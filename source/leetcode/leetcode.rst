=============================
LeetCode Challenge
=============================

这一章，我将在这里记录我在LeetCode上做题过程。我会在这里写下解题的思路，每一题都会有多个版本，供大家对比学习，真正做到一题多解，全方位理解题目的精髓。

网上的在线题库很多，比较出名的可能有 LeetCode 和 CodeWars ，这两个平台，我都有去注册尝试过。说实话，都很不错。

LeetCode 提交答案后，可以知道排名，让自己知道自己的代码处在什么样的层次。

而 CodeWars 是没有排名的，只要运行成功能取得正确答案就能通过，但是其有一个做得相当好的是，可以看到别人优秀的代码展示（人工点赞排名的）。这对于我们学习别人的代码，提供了很好的范本。



经过一系列的考量后，我还是选择大家熟知的 LeetCode ，因为这里的题目比较有挑战性，也比较权威，许多大公司都会在上面出题给求职者。
